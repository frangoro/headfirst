## Create runnable jar in Eclipse

1. right-click on my project in Eclipse
2. select "Export"
3. select "JAR file"
4. uncheck the top-level files like .classpath, .project
5. check only "export generated class files"
6. click "finish"

## Run the jar file

java -jar youJarFile.jar
